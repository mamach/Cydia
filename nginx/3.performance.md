# Performance

## Headers and Expires
* Expires header
```
Expires <time>;
```

## adding header
*
```
add_header my_header "Hello World";
add_header Pragma public;
add_header Cache-Control public;
add_header Vary Accept-Encoding;
expires 1M;
```
